EarthML¶

Machine learning and visualization in Python for Earth science

EarthML is an open-source Python library for exploratory machine learning research in Earth science.

We recommend the Getting Started Guide to learn the basic concepts and start using Project as quickly as possible.

The User Guide goes into key concepts more deeply when you are ready for further study.

The API is the definitive guide to each EarthML object.

Please feel free to report issues or contribute code.
